About

Zyrox is a userscript-based client enhancement for Gimkit. This page is kept intentionally simple so users can get started quickly.

  • Quick setup with Tampermonkey or Violentmonkey.
  • Uses the script in this repo: zyrox-base.js.
  • Clean, minimal layout focused on installation first.

Install

Follow these steps:

  1. Install a userscript manager extension (Tampermonkey or Violentmonkey).
  2. Open zyrox-base.js from this repository.
  3. Copy the script into your userscript manager and save it.
  4. Refresh Gimkit and confirm the script is enabled.
