Install

Follow these steps to install the userscript:

  1. Install a userscript manager browser extension, such as Tampermonkey or Violentmonkey.
  2. Open the script file from this repository: zyrox-base.js.
  3. Copy the script into your userscript manager and save it.
  4. Reload Gimkit and verify the script is enabled in your extension dashboard.

Optional: You can also add a direct install link here later once a hosted raw script URL is available.
